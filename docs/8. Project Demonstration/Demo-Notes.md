# Project Demonstration

## Demonstration Flow

1. Open the Library Management scoped application in ServiceNow Navigator
2. Log in as **Administrator** — show application menu, roles, and ACL configuration
3. Log in as **Librarian** — add a book, register a member, issue a book, process a return, view fine calculation
4. Log in as **Student** — search available books, view own issued books and fine status
5. Open the **Library Management Dashboard** — walk through Available Books, Book Issues, Fine Collection, and Total Members widgets
